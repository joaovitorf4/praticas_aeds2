# Implementação e Comparação: Árvore Binária de Pesquisa vs. Árvore AVL

Este repositório contém a 1ª Avaliação Prática da disciplina de Algoritmos e Estruturas de Dados II. O curso pertence ao Centro Federal de Educação Tecnológica de Minas Gerais (CEFET-MG) e é ministrado pelo Prof. Thiago de Souza Rodrigues.

## Descrição do Projeto

O objetivo do trabalho é implementar e avaliar o comportamento de uma Árvore Binária de Pesquisa Sem Balanceamento em comparação com uma Árvore AVL. As duas estruturas de dados devem armazenar unicamente elementos inteiros.

É necessário implementar os seguintes métodos para ambas as árvores:
* INSERÇÃO de um elemento.
* BUSCA de um elemento.

## Experimentos

A avaliação de desempenho consistirá na pesquisa pelo elemento 100.001 e na contagem do número de comparações executadas. Os testes devem ser feitos nos seguintes cenários:

### Árvore Binária de Pesquisa Sem Balanceamento
* Gerar 10 árvores a partir de n elementos ORDENADOS, com n variando de 10.000 até 100.000, com intervalo de 10.000.
* Gerar 10 árvores a partir de n elementos ALEATÓRIOS, com n variando de 10.000 até 100.000, com intervalo de 10.000.

### Árvore AVL
* Gerar 10 árvores a partir de n elementos ORDENADOS, com n variando de 10.000 até 100.000, com intervalo de 10.000.
* Gerar 10 árvores a partir de n elementos ALEATÓRIOS, com n variando de 10.000 até 100.000, com intervalo de 10.000.

## Relatório e Análise de Dados

O grupo deve redigir um relatório no Moodle contendo o nome dos membros. O documento precisa incluir os resultados extraídos dos experimentos no seguinte formato:
* Gráficos correlacionando n com o número de comparações, utilizando os dados das árvores geradas a partir de inserções ordenadas.
* Tabelas consolidadas com o número total de comparações registradas.
